Welcome to CSS

Here we will discuss the various types of selectors in CSS

Element Selectors

Selects elements by their name

In the HTML

<section> bla bla bla </section>

In the CSS

section { color: blue; }

Class Selector

Selects elements by their class

In the HTML

<section class="mark-me"> bla bla bla </section>

In the CSS

.mark-me { color: red; }

Id Selector

Selects elements by their Id

In the HTML

<section id="some-name"> bla bla bla </section>

In the CSS

#some-name { color: red; }

Universal Selector

Selects all elements

In the HTML

<main> <section> bla bla bla </section> </main>

In the CSS

* { color: red; }

Attribute Selector

Selects all elements with a specific attribute

In the HTML

<img src="image.png">

In the CSS

[src] { color: red; }

In the CSS - with specific attribute value

[src="image.png"] { color: red; }

Pseudo Selector

Selects all elements according to condition

In the HTML

<section> bla bla bla </section>

In the CSS

section:hover { color: red; }

Pseudo Element Selector

Selects Elements that are created dynamially by the selector

In the HTML

<section> bla bla bla </section>

In the CSS

section::first-letter { color: red; }

Combinators

Combine selectors to create new more specific selectors

In the HTML

<ul> <li> Item </li> <li> Item </li> <li> Item </li> </ul>

In the CSS

ul > li { color: red; } ul li { color: red; }
